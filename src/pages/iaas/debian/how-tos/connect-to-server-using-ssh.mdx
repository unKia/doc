# اتصال به سرور ابری Debian با استفاده از SSH
<hr className="mb-2" />

SSH (Secure Shell) یک پروتکل رمزنگاری‌شده برای اتصال امن به سرورهای ابری از طریق شبکه‌های ناامن، مانند اینترنت است. این پروتکل جایگزین امنی برای Telnet و سایر روش‌های قدیمی ورود به سرورها محسوب می‌شود، زیرا تمام داده‌های ارسالی و دریافتی را رمزنگاری می‌کند. SSH معمولاً برای مدیریت سرورهای لینوکسی، انتقال فایل (با استفاده از SCP یا SFTP) و اجرای دستورات از راه دور استفاده می‌شود.
